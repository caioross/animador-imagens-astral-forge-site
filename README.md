Landing bilíngue (PT/EN), explicando o pipeline local imagem→vídeo (Qwen2.5-VL + LTX-Video) que anima as artes de Astral Forge.

## Rodar localmente

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # produção
```

## Stack

Next.js 14 (App Router) · React 18 · TypeScript · Tailwind CSS · i18n client-side · Google Analytics 4.
